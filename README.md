# Spring-Data-Redis-Repositories

Work example with Redis Repositories, convert and store domain objects ind Redis Hashes (ie https://redis.io/topics/data-types).



Requirements
*********************

|  Tools    |     Version    
| ------------- |: -------------: | 
| Redis server      |        >= 2.8       |  
| Spring Data Redis         |        >= 2.O        | 
| Spring Boot     |        2.1.3        |  
| Jedis     |        2.9       |   
| java     |        1.8        |
